Cross-Platform AI Analysis Dataset

1) Master Dataset Purpose

This dataset is designed to allow three different AI systems to review the same underlying communication records and produce comparable outputs about:

  • communication patterns
  • escalation points
  • co-parenting coordination issues
  • emotional tone
  • child-centered recommendations
  • non-biased suggestions for improving the situation

5) Standard Analysis Prompt for All 3 Platforms

You are reviewing a structured communication dataset involving co-parenting and related family communications.

Your task is to provide a neutral, non-biased analysis focused on the best interest of the children.

Please analyze the dataset and provide the following:

1. Key communication patterns
2. Repeated misunderstandings or failure points
3. Escalation triggers
4. Signs of productive vs. unproductive communication
5. Whether communication appears centralized or fragmented
6. Potential risks to child stability caused by adult communication patterns
7. Non-biased recommendations for improvement
8. A child-centered communication strategy
9. A severity ranking of the top 5 communication issues
10. A concise executive summary

Important instructions:
- Do not take sides
- Do not make legal conclusions
- Do not assume facts not present in the data
- Distinguish clearly between facts, patterns, and interpretations
- Focus on practical recommendations that improve communication and reduce conflict for the children

Use this response structure:

A. Executive Summary
B. Key Patterns
C. Escalation Triggers
D. Child Impact Risks
E. Top 5 Communication Issues Ranked by Severity
F. Recommendations
G. Suggested Communication Protocol

6) Scoring Sheet for Easier Comparison Between Platforms

After completing the narrative analysis, score the following categories from 1 to 5:

1 = very low / minimal issue
5 = very high / severe issue

Categories:
- Communication fragmentation
- Misinterpretation risk
- Emotional escalation risk
- Child-centeredness of communication
- Clarity of logistics communication
- Presence of triangulation risk
- Transparency between adults
- Overall communication health

Then provide a brief reason for each score.

8) Best Practices for Fair Cross-Platform Analysis

  • use the same exported data
  • use the same cleaned dataset
  • use the same prompt
  • use the same output template
  • do not add extra commentary to one platform and not the others
  • keep names and thread labels consistent
  • redact sensitive info the same way across all versions
