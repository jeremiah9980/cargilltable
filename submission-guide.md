# How to Submit Data to AI Platforms

Follow the same steps for each platform to keep the analysis consistent.

## Step 1 — Prepare dataset

Fill these files:

- participants_master.csv
- conversations_master.csv
- messages_master.csv
- incidents_master.csv

## Step 2 — Create packet

Move the normalized data into:

packet_common.json

## Step 3 — Copy standard prompt

Paste the project prompt and scoring instructions.

## Step 4 — Submit to platforms

Submit the same packet to:

- ChatGPT
- Grok
- Gemini

## Step 5 — Save outputs

Save each response into:

- chatgpt_output.md
- grok_output.md
- gemini_output.md

## Step 6 — Compare analysis

Compare:

- communication risks
- triangulation findings
- role classifications
- recommendations
